To run this Shell Simply Run 

```python3
python3 mysh.py
```
OR 
```python3
python3 mysh.py test
```


Usage:
```
$ say cheese
cheese
$ say \
> cheese
cheese
$ say \
> blue \
> cheese
blue cheese
$ exit
Goodbye!
```
```
$ set a alpha
$ say $a
alpha
$ set b bravo
$ say $a $b charlie
alpha bravo charlie
$ set
a=alpha
b=bravo
PS=$
$ exit
Goodbye!
```
